# EthFund

このDappはEthFundです。分散型クラウドファンデーションアプリです。この分散型クラウドファンデーションアプリを作った理由は、ブロックチェーン技術をつかうことによって、仲介人をなくし、中間マージンを抑えられるからです。
例えば、キックスターターでは5%の手数料がかかり、チャレンジャーが獲得したお金からひかれてしまいます。
